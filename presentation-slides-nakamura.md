# AIとの付き合い方

## AIは **大きい作業が苦手**

### タスクは細分化する

### ❌ 大きすぎる指示

```
「同窓会の運営よろしく」
```

↓ AIは困る 🤷

---

## こう分ける

### ✅ 小さく、さらに小さく

```
同窓会の運営よろしく
├── 📋 会員名簿を更新
│   ├── 住所変更を反映して
│   ├── 重複データを削除して
│   └── 連絡先不明者をリストアップして
├── 📝 案内状を作成
│   ├── 日時・場所を記載して
│   ├── 出欠の返信方法を書いて
│   └── 会費の振込先を追記して
├── 📊 出欠を集計
└── 🍽️ 会場に連絡
```

→ **分解するほどAIは賢く動く**

# 地域社会のAI導入

## AIは魔法じゃない

AIが動くには **データ** が必要

```
紙の会報 → AIは読めない
PDFの画像 → AIは読めない
整理されたテキスト → ✅ AIが活用できる
```

→ まず **デジタル化・構造化** から

→ 情報を使える状態にする、オープンにする

---

# よくある現状（Before）

## ❌ こうなってませんか？

- 連絡は **LINEグループ** でバラバラ
- 資料は **Word → PDF** にしてメール添付
- 各自のPCにダウンロード、どれが最新か不明
- 「あの件どうなった？」→ 過去トークを遡って探す
- 打ち合わせは **電話か対面**、記録なし
- 新しい人が入ると **一から説明** が必要

→ 情報が **散らばって埋もれている**

---

# 整理するとこう変わる（After）

## ✅ AI時代の情報管理例

- ドキュメントは **Notion / Google Drive / Git** で一元管理
- やり取りは **Slack** でスレッド化・検索可能
- 会議は **Meet** → Geminiで **自動議事録**
- 対面MTGも **Plaud** で録音 → AI文字起こし
- AIが過去の情報を **横断検索**
