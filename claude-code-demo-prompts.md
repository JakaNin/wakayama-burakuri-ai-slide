# Claude Code デモ用プロンプト集

## 概要

このドキュメントは、プレゼンテーション「デモ②：Claude Codeで編集」で使用するプロンプト集です。
既存サイト https://dousoukai.site/kourokai/ の要素を kouro-lp に追加するデモを行います。

## 使い方のヒント

### デモの流れ

1. **パターンA** から始める（簡単さをアピール）
2. **パターンB** で機能追加の威力を見せる
3. **パターンC** で「ここまでできる」を示す

### 時間の目安

| パターン | 所要時間 | 聴衆へのインパクト |
|:--|:--:|:--:|
| A（シンプル編集） | 10-30秒 | ★★☆ |
| B（機能追加） | 1-2分 | ★★★ |
| C（新規ページ/大規模変更） | 2-5分 | ★★★★ |

### 注意点

- デモ前に `npm run dev` でサーバーを起動しておく
- ブラウザのホットリロードで即座に変更が反映される
- エラーが出た場合も「AIが自動修正する様子」を見せられる
