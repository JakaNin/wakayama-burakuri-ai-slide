- v0.dev にログイン済み
- プロンプトは事前にメモ帳に用意
- 生成結果は事前テストで確認済み（バックアップとして保存）

#### トーク

```
「まず使うのは、v0（ブイゼロ）というサービスです。

では早速やってみましょう。」
```

#### 画面操作
1. v0.dev を開く
2. プロンプトを入力（事前準備したものをコピペ）

```
【プロンプト例】
和歌山県の観光情報サイトのランディングページを作成してください。
- ヘッダーに「Discover Wakayama」のタイトル
- 熊野古道、高野山、白浜の3つの観光スポットをカード形式で紹介
- 各カードには画像プレースホルダー、タイトル、短い説明文
- お問い合わせボタン
- モダンで清潔感のあるデザイン
- 日本語で
```

3. 生成を待つ（この間にトーク）

```
「今、AIがデザインを考えています。
従来であれば、デザイナーに依頼して、ワイヤーフレームを作って、
デザインカンプを確認して...という作業で1〜2週間。

それが今、数十秒で...」

```

| 修正対応 | 都度見積もり | その場で即対応 |

### 【バッファ】20:00-25:00（5分）

想定される質問と回答：

**Q: 本当に素人でもできますか？**
```
「今日お見せしたレベルであれば、できます。
ただ、最初は少し練習が必要です。
ChatGPTを普段から触っていると、コツがつかめてきます。」
```

**Q: セキュリティは大丈夫ですか？**
```
「今日作ったような公開情報のサイトであれば、特に問題ありません。
ただし、顧客情報や社内機密を扱うシステムは、
別途セキュリティの専門家に相談することをお勧めします。」
```

**Q: コストは本当にそれだけですか？**
```
「v0は月額20ドル程度、Claude Codeも同程度です。
もちろん、運用するサーバー代などは別途かかりますが、
開発費という意味では、従来の10分の1以下になることが多いです。」
```

**Q: どのAIツールがおすすめですか？**
```
「用途によります。
文章やアイデア出しならChatGPT、
コードを書くならClaude、
Webデザインならv0、
という使い分けをしています。
まずはChatGPTから始めるのがおすすめです。無料で使えますので。」
```
